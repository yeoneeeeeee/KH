flexBox

        요소의 정렬되는 방향, 순서, 요소간의 간격을 수치적으로 처리하지 않아도
        알아서 잘 유연(flex)하게 처리해주는 형식


        * flexbox를 이용할때 반드시 알아야 되는것

        1. FlexBox의 구성
        - flex-container : 정렬이 필요한 요소들을 감싸는 요소 
        - item : 정렬을 적용할 요소
        (flex-container와 item에 사용되는 flex관련 css속성이 나누어져 있다.)

        2. flexbox의 축
        - 중심축(main axis)
        - 교차축(cross axis)
